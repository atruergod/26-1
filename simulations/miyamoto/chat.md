## Q1–Q7. 논문 분석

### 1. 논문 요약

이 논문은 한국 대학에 재학 중인 외국인 유학생(86명)의 **한국어 쓰기 태도**를 학습자 요인별로 분석한 실증 연구입니다.

**배경**: Study Korea Project 정책으로 외국인 유학생이 증가함에 따라 학술적 글쓰기 능력이 중요해졌으나, 쓰기의 인지적 영역 연구에 비해 정의적(affective) 영역인 쓰기 태도 연구는 크게 부족했습니다.

**연구 설계**:
- 오택환(2009)의 도구를 재수정한 21문항 Likert 5점 척도 설문
- 쓰기 태도를 세 하위 범주로 구분: **쓰기인식**(필요성·가치 인식), **쓰기반응**(흥미·자신감), **수행태도**(내용 생성·표현·조직·고쳐쓰기)
- 타당도(KMO=.756, Bartlett p=.000), 신뢰도(Cronbach α=.854) 검증 완료

**연구 참여자(N=86)**:
- 성별: 여학생 59명, 남학생 27명
- 학년: 1학년 54명, 2학년 8명, 3학년 9명, 4학년 9명, 교환학생 6명
- 국적: 미얀마(25), 몽골(15), 베트남(14), 중국(12), 우즈베키스탄(6) 등 13개국
- 한국어 숙달도(TOPIK): 중급(3·4급) 52명, 고급(5·6급) 34명
- 어학원 경험: 유경험자 69명, 무경험자 17명

**분석 방법**: 기술통계, 독립표본 t-검정, 일원배치 분산분석(One-way ANOVA), Pearson 상관분석(SPSS Ver.22)

**주요 결과**:
- 전체 쓰기 태도 평균: M=3.511
- 하위 범주 평균: 쓰기인식(4.015) > 수행태도(3.511) > 쓰기반응(3.348)
- 학년, 성별, TOPIK, 어학원 경험 등 모든 학습자 요인과 쓰기 태도 간에 유의미한 차이 없음
- 상관분석: 성별↔쓰기인식(r=.290\*\*), 학년↔수행태도(r=.328\*\*)에서만 약한 정적 상관

**결론**: 한국어 숙달도나 어학원 학습 경험이 높아져도 긍정적 쓰기 태도는 자동으로 형성되지 않으므로, 교수자·연구자가 정의적 요인 함양에 별도의 관심을 기울여야 한다.

### 2. 핵심 연구 질문과 결과

**연구 질문**: 외국인 유학생의 한국어 쓰기 태도는 학습자 요인(학년, 성별, 한국어 숙달도, 어학원 경험)에 따라 유의미하게 다른가?

|---|---|---|
| 학년 | 일원배치 ANOVA | 수행태도만 유의(p<.05), 쓰기인식·반응은 비유의 |
| 성별 | 독립표본 t-검정 | 비유의 |
| 한국어 숙달도(TOPIK) | 독립표본 t-검정 | 비유의 |

### 3. 논문의 학문적 기여

1. **연구 공백의 개척**: 한국어교육 분야에서 쓰기 태도를 다룬 연구는 곽준화(2018) 단 한 편뿐이었던 공백을 실증적으로 채운 최초의 본격적 시도
2. **정의적 영역의 독립성 규명**: 언어 숙달도(인지적 요인)와 쓰기 태도(정의적 요인)가 별개로 작동함을 통계적으로 확인
3. **교육적 함의 제공**: 장르 기반·기능 중심 쓰기 교육만으로는 정의적 태도 개선이 어렵다는 증거 제시
4. **기초 자료 축적**: 외국인 유학생의 쓰기 태도 현황을 최초로 수량화하여 후속 연구를 위한 기준치 제공

### 4. 논문의 강점과 약점

**강점**
- 측정 도구의 심리측정학적 검증(KMO, Bartlett, Cronbach α)을 체계적으로 수행
- 복수의 학기에 걸쳐 자료를 수집함으로써 계절 효과를 부분적으로 통제
- 정의적 영역 연구의 필요성을 실증적으로 환기

**약점**
- **표본 크기(N=86)가 소규모** — 통계적 검정력(statistical power) 낮음
- **국적 변인의 미활용**: 13개국 출신이 혼재함에도 국적을 분석 변인에서 제외
- **단일 시점 횡단 연구**: 쓰기 태도의 변화 과정 추적 불가
- **자기보고식 설문의 한계**: 사회적 바람직성 편향 통제 장치 없음
- **집단 크기의 불균형**: 1학년 54명 vs. 2학년 8명

### 5. '국적' 변인 미활용 이유와 활용 가능성

**왜 제외되었는가?**  
13개 국적 집단 중 분석 가능한 최소 셀 크기를 갖춘 집단이 소수에 불과하며, 86명을 13개 셀로 분할하면 셀당 평균 6.6명으로 통계적 검정력이 극히 낮습니다.

**국적을 활용했다면 가능한 분석**:
- 언어적 거리(language distance) 관점: 한국어와 계통적으로 가까운 집단 vs. 먼 집단 비교
- 유교 문화권 vs. 비유교 문화권 학습자의 쓰기 인식 차이
- 출신국 교육 체계(쓰기 중심 vs. 구술 중심)가 수행태도에 미치는 영향

**추가로 유용한 피험자 정보**: 전공 계열, 모국어 표기 체계, 한국 체류 기간, 한국어 학습 동기, 이전 L2 쓰기 경험, 쓰기 불안(writing anxiety) 점수, 장학금 유형 등

## 석사논문 주제 적절성 평가

### 주제 1. 어권별(한자권 vs 비한자권) 비교 ★★★☆☆

가장 큰 문제는 **범주 설정의 타당성**입니다. 베트남은 역사적으로 한자 문화권에 속하며, 미얀마와 유럽을 같은 '비한자권'으로 묶는 것도 언어학적으로 근거가 취약합니다.

**개선 방향**: '한자권 vs 비한자권' 이분법 대신 '언어적 거리'를 연속 변인으로 설정하고, 새로운 표본(최소 200명 이상)을 모집하여 다중회귀 또는 다층 모형으로 분석.

### 주제 2. 대학 특성별(종합대 vs 이공계 특성화대) 비교 ★★★★☆

가장 현실적이고 독창적인 주제. 기존 연구의 한계(단일 기관)를 직접 보완하며 외적 타당도를 높입니다. 두 유형의 대학에서 각각 100명 내외를 모집하면 충분한 검정력 확보 가능.

**주의**: KAIST, UNIST의 학문 목적 한국어 학습자 비율이 낮을 수 있어 사전 모집 가능 인원 확인 필요.

### 주제 3. 교육 과정별 쓰기 태도 종단 연구 ★★★★★

학문적 가치 최고, 단 실행 난이도도 최고. '어학원 경험이 쓰기 태도에 무의미하다'는 결론이 교육과정 질적 차이를 무시한 것임을 검증하는 강력한 후속 연구.

**현실적 어려움**: 종단 연구는 중도 탈락률이 높고, 두 기관의 협조가 필요하며, 석사 2년 안에 완성하기 매우 도전적.

### 주제 4. 쓰기 태도와 학업 성취도(GPA) 상관성 ★★★☆☆

정의적 요인 함양의 실질적 효용을 증명하려는 시도. **GPA 데이터 접근이 가장 큰 현실적 장벽**. 개선 방향: GPA 대신 연구자가 직접 채점을 통제하는 쓰기 성취도 과제를 종속변인으로 설정.

### 주제 5. 생성형 AI 활용과 쓰기 태도 ★★★★☆

시의성·파급력 최고. AI 도구 사용이 쓰기 효능감을 높이는지 혹은 쓰기인식을 저해하는지 탐색. 이론적 틀(TAM, 정의적 필터 가설) 정립이 선행 과제.

## Stan/CmdStanPy 기반 차별화 연구 주제 제안

### 주제 1. 계층적 베이즈 모형 (Hierarchical Bayesian Model)

**핵심**: 부분 풀링(partial pooling)으로 소규모 국적 집단도 안정적 추정.

```stan
// 국적 랜덤 효과 + 구조 모형 예시
model {
  mu_global ~ normal(3.5, 1.0);
  sigma_country ~ exponential(2);
  for (j in 1:J)
    alpha_country[j] ~ normal(0, sigma_country);
  to_vector(beta) ~ normal(0, 0.5);
  for (n in 1:N)
    for (k in 1:K)
      y[n,k] ~ normal(mu_global[k] + alpha_country[country[n], k]
                      + X[n] * beta[,k], sigma_resid[k]);
}
```

### 주제 2. 순서형 베이즈 회귀 (Ordinal Bayesian Regression)

**핵심**: Likert 5점 척도를 연속형으로 오취급하는 기존 분석의 근본 오류 해결.

```stan
parameters {
  ordered[4] c;  // 절단점 — c[1] < c[2] < c[3] < c[4] 자동 보장
  vector[P] beta;
}
model {
  for (n in 1:N)
    for (k in 1:K)
      y[n,k] ~ ordered_logistic(X[n] * beta + alpha[country[n]], c);
}
```

### 주제 3. 베이즈 혼합 모형 (Bayesian Mixture Model)

**핵심**: 학습자를 "의욕형", "회피형" 등으로 유형화하고 LOO-CV로 최적 군집 수 결정.

### 주제 4. 상태공간 모형 (State-Space Model)

**핵심**: 종단 연구에서 중도 탈락을 자연스럽게 처리하고 태도 변화 궤적 추정.

### 주제 5. 베이즈 SEM (BSEM)

**핵심**: 쓰기태도 → AI활용 → 쓰기성취도 경로의 매개 효과를 완전한 사후 분포로 추정.

### ✅ 올바른 부분

- X→M→Y 부분 매개 모형 구조의 이론적 타당성 ✓
- 간접 효과 = β₁ × β₂, 총 효과 = γ₁ + β₁β₂ 계산식 정확 ✓
- PCM 임계값(thresholds)의 비순서(disordered) 허용은 PCM에서 맞음 ✓
- Q6 수식 ↔ Q7 Stan 코드의 수학적 일치 ✓
- PCM 누적 log-prob 공식의 올바른 구현 ✓

### 🔴 심각한 오류 (Critical)

**오류 1 — 내생 잠재변수(M, Y)의 척도 비식별**

`theta[n][1] ~ normal(0, 1)` 로 X의 식별은 확보되었으나, M과 Y의 잔차 분산 σ_M, σ_Y가 자유 파라미터로 남아 있어 `(β₁, σ_M, δ^(M))` 방향의 ridge가 존재합니다.

**오류 2 — Stan 구식 배열 문법 (컴파일 오류)**

```stan
// ❌ 구식 문법 (Stan 2.26+ 에서 deprecated)
int<lower=1, upper=K> y[N, I];
vector[K-1] delta[I];
vector[3] theta[N];

// ✅ 현대 Stan 문법
array[N, I] int<lower=1, upper=K> y;
array[I] vector[K-1] delta;
array[N] vector[3] theta;
```

**오류 3 — Stan 루프 내 로컬 변수 선언 순서**

```stan
// ❌ 실행문 이후 변수 선언
for (i in 1:I) {
  int latent_idx;
  if (i <= 4) latent_idx = 1;  // 실행문
  vector[K] probs;             // 실행문 이후 선언 → 오류
  vector[K] log_probs;
}
```

### 🟡 중간 수준 오류 (Moderate)

- **오류 4**: 미사용 변수 `vector[K] probs` 선언
- **오류 5**: Q4의 파라미터 수 359 vs. Q6 단순화 모형 349 불일치

### 🔵 경미한 오류 (Minor)

- **오류 6**: "분산(variance)" vs. "표준편차(standard deviation)" 용어 혼용
- **오류 7**: X 식별 조건을 "고정"이라고 표현하였으나 엄밀히는 사전 분포 부여

## 오류 1 수학적 해설: σ_M = 1 고정의 근거

### 위치 비식별의 형식적 구조

임의의 상수 $c$에 대해 $M_n' = M_n + c$, $\delta_{ij}^{(M)'} = \delta_{ij}^{(M)} + c$ 로 놓으면:

$$M_n' - \delta_{ij}^{(M)'} = M_n - \delta_{ij}^{(M)}$$

PCM 로그-오즈 불변 → 두 파라미터 집합은 관측적으로 동치 → **위치 비식별 존재**.

### 동치 파라미터 군(Equivalence Class)

구조 방정식에서 상수 $c > 0$에 대해:

- $\boldsymbol{\phi}_A$: $(β_1, γ_M, σ_M, \{δ_{ij}^{(M)}\})$
- $\boldsymbol{\phi}_B$: $(cβ_1, cγ_M, cσ_M, \{cδ_{ij}^{(M)}\})$

$\boldsymbol{\phi}_B$ 하에서 $M_n^{(B)} = c \cdot M_n^{(A)}$. PCM 적용 시:

$$k \cdot cM_n^{(A)} - c\sum_{j=1}^k δ_{ij}^{(M)} = c\\!\left(kM_n^{(A)} - \sum_{j=1}^k δ_{ij}^{(M)}\right)$$

$c \neq 1$이면 우도가 달라지므로 **척도는 PCM에서 식별됩니다**. 그러나 실용적으로 $(β_1, σ_M, δ^{(M)})$ 사이의 ridge-like geometry가 존재하여 MCMC 수렴을 방해합니다 — **Neal's Funnel 현상**.

### σ_M = 1 고정의 효과

파라미터 공간에서 ridge를 하나의 점으로 붕괴:

$$\{(cβ_1, cσ_M, cδ^{(M)}) : c > 0\} \xrightarrow{σ_M = 1} \left(\frac{β_1}{σ_M}, 1, \frac{δ^{(M)}}{σ_M}\right)$$

경로 계수 해석: "X가 1 SD 증가할 때, M의 구조적 평균이 $β_1$ 단위 증가한다. M의 단위는 잔차 표준편차 = 1로 정규화된 log-odds."

### 절편 + 합-영 제약으로 위치 비식별 해결

**합-영 제약**: $\sum_{i \in M\text{-subscale}} \sum_{k=1}^{K-1} δ_{ik}^{(M)} = 0$

위치 불변 변환 시도: $\sum_{i,k}(δ_{ik}^{(M)} + c) = 0 + I_M(K-1) \cdot c = 0 \Rightarrow c = 0$ ✓

**∴ σ_M = 1 (척도) + α_M 절편 + 합-영 제약 (위치) = 완전 식별**

## 완성된 Stan 코드 (sem_pcm_v2.stan)

```stan
// sem_pcm_v2.stan
// ┌─────────────────────────────────────────────────────────────┐
// │  PCM-SEM 통합 모형 — 완전 식별 버전                           │
// │  [척도 식별] σ_M = σ_Y = 1 고정                              │
// │  [위치 식별] α_M, α_Y 명시적 절편                             │
// │            + M·Y 하위 척도 합-영(sum-to-zero) 제약            │
// └─────────────────────────────────────────────────────────────┘

data {
  int<lower=1> N;
  int<lower=1> I;
  int<lower=1> K;
  int<lower=1, upper=I-1>   I_X;
  int<lower=1, upper=I-I_X> I_M;
  array[N, I] int<lower=1, upper=K> y;
  vector[N] gender;
}

transformed data {
  int I_Y = I - I_X - I_M;
}

parameters {
  array[I] vector[K-1] delta_raw;
  array[N] vector[3]   theta;
  real b1;       // β₁ : X → M
  real b2;       // β₂ : M → Y
  real g1;       // γ₁ : X → Y (직접 효과)
  real gamma_M;
  real gamma_Y;
  real alpha_M;  // M 절편
  real alpha_Y;  // Y 절편
}

transformed parameters {
  array[I] vector[K-1] delta;

  for (i in 1:I_X)
    delta[i] = delta_raw[i];

  {
    real M_offset = 0.0;
    for (i in (I_X + 1):(I_X + I_M))
      M_offset += sum(delta_raw[i]);
    M_offset /= (I_M * (K - 1));
    for (i in (I_X + 1):(I_X + I_M))
      delta[i] = delta_raw[i] - M_offset;
  }

  {
    real Y_offset = 0.0;
    for (i in (I_X + I_M + 1):I)
      Y_offset += sum(delta_raw[i]);
    Y_offset /= (I_Y * (K - 1));
    for (i in (I_X + I_M + 1):I)
      delta[i] = delta_raw[i] - Y_offset;
  }
}

model {
  for (i in 1:I)  delta_raw[i] ~ normal(0, 3);
  b1      ~ normal(0, 1);
  b2      ~ normal(0, 1);
  g1      ~ normal(0, 1);
  gamma_M ~ normal(0, 1);
  gamma_Y ~ normal(0, 1);
  alpha_M ~ normal(0, 1);
  alpha_Y ~ normal(0, 1);

  for (n in 1:N) {
    theta[n][1] ~ normal(0, 1);
    theta[n][2] ~ normal(
        alpha_M + b1 * theta[n][1] + gamma_M * gender[n], 1.0);
    theta[n][3] ~ normal(
        alpha_Y + g1 * theta[n][1] + b2 * theta[n][2]
        + gamma_Y * gender[n], 1.0);

    for (i in 1:I) {
      int       latent_idx;
      vector[K] log_probs;

      if      (i <= I_X)           latent_idx = 1;
      else if (i <= I_X + I_M)     latent_idx = 2;
      else                         latent_idx = 3;

      log_probs[1] = 0;
      for (k in 2:K)
        log_probs[k] = log_probs[k-1]
                     + (theta[n][latent_idx] - delta[i][k-1]);

      y[n, i] ~ categorical_logit(log_probs);
    }
  }
}

generated quantities {
  real indirect_effect = b1 * b2;
  real total_effect    = g1 + indirect_effect;
  real prop_mediated   = (fabs(total_effect) > 1e-10)
                         ? indirect_effect / total_effect
                         : not_a_number();
}
```

## Python 실행 스크립트 (run_sem.py)

```python
import numpy as np
import pandas as pd
import arviz as az
from cmdstanpy import CmdStanModel

y_matrix      = df[[f'y{i}' for i in range(1, 22)]].values.astype(int)
gender_vector = df['gender'].values.astype(float)

stan_data = {
    'N': len(df), 'I': 21, 'K': 5,
    'I_X': 4, 'I_M': 11,
    'y': y_matrix, 'gender': gender_vector,
}

model = CmdStanModel(stan_file='sem_pcm_v2.stan')

fit = model.sample(
    data=stan_data,
    iter_warmup=1000, iter_sampling=2000,
    chains=4, parallel_chains=4,
    seed=42, adapt_delta=0.95, max_treedepth=12,
)

idata = az.from_cmdstanpy(fit)
key_params = ['alpha_M', 'alpha_Y', 'b1', 'b2', 'g1',
              'gamma_M', 'gamma_Y',
              'indirect_effect', 'total_effect', 'prop_mediated']
print(az.summary(idata, var_names=key_params, round_to=3))

# 합-영 제약 검증
delta_samples = fit.stan_variable('delta')
for (start, end), name in [((4, 15), 'M (반응)'), ((15, 21), 'Y (수행)')]:
    sums = delta_samples[:, start:end, :].sum(axis=(1, 2))
    print(f"{name} delta 합: mean={sums.mean():.8f}, std={sums.std():.8f}")
```
